## 👨‍💻 About Me

I'm a **9th-grade student** with a passion for full-stack development and DevOps. I started coding to build useful things, and now I'm turning that passion into real projects.

- 🔭 **Currently building:** [SafeWire](https://github.com/AlmazIsDev/SafeWire) — A massive messenger with React (frontend), Python (backend), and React Native + Java (mobile app). Started in July 2025.
- 🚀 **Past projects:** 
  - **WeissX Hosting** (Dec 2024 - Jun 2025, Aug 2025 - Dec 2025) — My own hosting service (now closed)
  - **MXL Hosting** (May 2025 - present) — Active hosting service
  - **Tech-Developers** — Website development company
- 🤖 **Bot:** [MAX--TG](https://github.com/AlmazIsDev/MAX-TG) — Forwards messages from messenger to Telegram
- 🌱 **Learning:** Deepening my skills in FastAPI, React Native, and exploring cloud DevOps
